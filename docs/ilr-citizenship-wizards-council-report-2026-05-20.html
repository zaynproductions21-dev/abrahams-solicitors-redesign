Advisor B · First Principles
Question every assumption
Approve with changes

Symmetry is wrong. ILR is a richer multi-route decision; Citizenship is a gating check on top of ILR. Force-fitting 6Q on both creates false confidence on ILR and padding on Citizenship. Same engine, variable length.

  1. ILR 8–9Q; add dependants, prior refusals, continuous lawful gap.
  2. Citizenship 4–5Q; drop redundancies implied by ILR.
  3. Separate URLs (Placement B).
  4. Confidence-level badge on result.

Chairman Synthesis — Recommended Changes

Engine Refactor (Before Any Wizard #2 Work)

  1. Refactor VisaWizardWidget to accept a config prop: { questions[], routes[], scoringFn, wizardType, lastReviewed, author }.
  2. Add wizard_type parameter to all GTM events.
  3. Support variable question count from day 1.

ILR Wizard (8–9 Questions)

  1. Add Q0: "Have you switched visa categories in the last 5 years?" → solicitor route if yes.
  2. Add: dependants, prior ILR refusals, continuous lawful status gap.
  3. Any "not sure" answer → force solicitor route. No pathway shown.
  4. Confidence-level badge on result card (high / medium / low).

Citizenship Wizard (4–5 Questions)

  1. Drop redundancies implied by holding ILR (English B1 + Life in UK Test already evidenced at ILR stage — verify with Imran).
  2. Remove "British by birth" option; link to gov.uk check tool.
  3. Same "not sure" → solicitor route.

Language & UX (Outsider Pack)

  1. 10-sec "Which wizard?" chooser at top of /visa-wizard/: "Can you stay forever? (ILR) / Can you get a passport? (Citizenship) / Bringing a partner? (Spouse Visa)".
  2. Hover-tooltips on every acronym: EUSS, ILR, B1, s.6(1), CEFR, Appendix FM.
  3. Plain-English helper under each question (e.g. "Life in UK Test = 24-question online quiz, £50").
  4. Rename result routes — "s.6(1)" → "Standard citizenship route"; "Continuous residence broken" → "Long trips may have reset your clock — let's check".
  5. "Not sure? That's normal — we'll explain." reassurance copy.

Safety & SRA Hygiene

  1. Visible disclaimer above every result: "Indicative only. Not legal advice. Eligibility confirmed only after document review."
  2. Log every routing decision (anonymised) for 6 years — SRA evidence trail.
  3. Imran personally signs off route logic at launch and reviews quarterly. "Last reviewed" date stamped on every result card.

Light-Touch Upside (v1.1)

  1. Opt-in "Email me a personalised PDF checklist + timeline" at result step.
  2. CRM next_trigger_date tag for cross-wizard re-engagement (ILR-eligible today → Citizenship reminder at +12 months).
